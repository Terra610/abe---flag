A.B.E. System Map

This is the high-level wiring of the American Butterfly Effect engine. It shows how CIRI (impact math), CIBS (reinvestment allocation), and Integration all connect into a verifiable, auditable recovery pipeline.

Core Components

  • CIRI — Constitutional Integrity ROI Engine
    Path: /ciri/
    Calculates recoverable value by preventing or repairing unconstitutional harm.
  • CIBS — Community Integrity Budget System
    Path: /cibs/
    Turns that recovered value into transparent, scheduled spending for housing, stability, re-entry, youth, etc.
  • Integration Layer
    Path: /integration/README.md
    Defines how the money moves and how each dollar is justified.

Raw Map JSON

For auditors / investigators / devs: the structured machine-readable map is here: 

system/map.json

That JSON declares version, module names, file paths, and purpose. It’s your public contract: “this is what we’re doing, this is where it lives, this is not for sale.”
