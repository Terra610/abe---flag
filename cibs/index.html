CIBS (Community Integrity Budget System)

The A.B.E. framework for transparent, ethical reinvestment of recovered funds.

💰 Using CIRI Outputs

When you run the CIRI calculator, it automatically generates a file called cibs/auto_budget.csv. This file represents your community’s Recovery Pool — the total financial value gained by restoring constitutional compliance.

CIBS divides that Recovery Pool into targeted investment categories such as:

  • 🏠 Housing Stabilization / Emergency Housing
  • 🎓 Youth Support / Re-entry / Aftercare
  • 🪖 Veterans & Emergency Relief
  • ⚖️ Legal Defense / Rights Protection
  • 🧠 Mental Health / Stability Access
  • 📋 Civic Compliance & Administration (Audit / Reporting)

The auto-budget file assigns a percentage and dollar value to each category. Users can open or edit it in any spreadsheet program for presentation to councils, courts, or the press.

📊 Manual Budgeting Option

If you prefer to work manually, open budget_template.csv and enter your own totals from CIRI or any verified audit. Use model.md to view or modify the percentage allocation rules.

🌍 Purpose

CIBS ensures every recovered dollar is publicly traceable from source to outcome. It replaces opaque “settlements” with transparent reinvestment — building trust, stability, and real economic recovery in the communities that were harmed.
