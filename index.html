How to Use the A.B.E. Models

The A.B.E. site is not just theory. You can run it. You can take it into court, budget hearings, negotiations, or press. Here's how it works in practice:

  1. Run CIRI (ROI Engine)
    Open ciri/inputs.csv and enter your local data: dismissed cases, jail-days avoided, fees canceled, licenses restored, etc. Then run:
    python3 ciri/calculate.py
    This prints a full impact report and produces a Recovery Pool dollar amount.
  2. Apply CIBS (Budget System)
    The script also writes cibs/auto_budget.csv. That file shows how the Recovery Pool must be reinvested: housing stability, youth, veterans, legal defense, mental health, audit compliance. This is the public demand document.
  3. Verify on the System Map
    Go to system/map.json. That file is the declared flow of money and accountability. If what leaders claim doesnâ€™t match that map, you can prove it.
  4. Use the Integration Layer for escalation
    If money disappears, reporting stops, retaliation starts, or cases get buried, the Integration Layer exists to escalate that pattern to ethical security/audit networks instead of letting it die quietly.

This is designed so that a public defender, a civil rights lawyer, a city council budget hawk, or a survivor can all use the same language and numbers.
